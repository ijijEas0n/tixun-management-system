# 径技与君体训管理系统

用于体育训练成绩录入、排名查看和学生档案管理的前端系统。

## 本地运行

1. 安装依赖：

   ```bash
   npm install
   ```

2. 启动本地服务：

   ```bash
   npm run dev
   ```

3. 打开浏览器访问：

   ```text
   http://localhost:3000/
   ```

## 发布方式

项目已配置 GitHub Pages 自动发布。

代码推送到 `main` 分支后，GitHub Actions 会自动执行检查、构建，并把 `dist` 发布到 GitHub Pages。

## 常用命令

```bash
npm run lint
npm run build
```

## 语音识别使用方式

语音录入使用腾讯云实时语音识别 Web SDK。

每台电脑第一次使用时，在“成绩录入”的语音录入栏点击“设置”，填写自己的腾讯云 `AppId`、`SecretId`、`SecretKey` 后保存。设置会保存在当前浏览器本地，不会写入项目代码或打包文件。

打包给别人使用时，对方只需要在自己的浏览器里填写自己的腾讯云信息即可，不需要你的电脑一直运行服务。

腾讯云侧需要开通“实时语音识别”，并确保中国大陆实时流量可用。
