# 径技与君体训管理系统

用于体育训练成绩录入、排名查看和学生档案管理的前端系统。

## 本地运行

1. 安装依赖：

   ```bash
   npm install
   ```

2. 启动本地服务：

   ```bash
   npm run dev
   ```

3. 打开浏览器访问：

   ```text
   http://localhost:3000/
   ```

## 发布方式

项目已配置 GitHub Pages 自动发布。

代码推送到 `main` 分支后，GitHub Actions 会自动执行检查、构建，并把 `dist` 发布到 GitHub Pages。

## 常用命令

```bash
npm run lint
npm run build
```
